GitHub Resume

This is a list of projects I have worked on. Most have been completed in my free time. More details about
any of the projects can be found in the README.md files of their GitHub repos. My email address is provided on my resume - if you have any questions, comments, or otherwise want to get in touch, I would be happy to hear from you.

Projects:

  • ErgCalc
    • iOS app that allows users to convert ergometer data between various formats. Originally designed for personal use after finding existing solutions lacking.
    • GitHub
  • Snap Text Effects
    • iOS app that allows users to create text with gradients or rainbows for use in Snapchat. This application became quite popular after being featured on a number of videos. It is currently on around 70,000 lifetime downloads. This app relied on a quirk in how Snapchat handles pasted text, which I believe has now been patched in later updates. As such, I have removed this app from the App Store as of March 2018.
    • GitHub
  • Computing Topics Project
    • First year project for Computing Topics 164. The project involved creating an informative webpage about a particular topic in Computer Science. Our project focused on In-Memory Database Systems, and in particular programming techniques to optimise their performance. I led this group project and did most of the web design.
    • Finished Project
  • Row Around The World
    • A web app which allows users to visualise how many meters they are rowing on a world map. Start and end markers can be set, and then meters rowed can be entered. This was created in order to make long erg sessions (slightly) less monotonous.
    • GitHub Live Demo
  • Green & Jenks Point-Of-Sale System
    • A proof-of-concept till system designed for a local ice cream shop. The till system had a fully featured GUI with a number of more advanced features, such as the ability to dynamically add new item buttons and change prices. The project makes use of a database to store item data, and uses some more advanced features of VB like generic types.
    • GitHub
  • Voler.io clone
    • A clone of voler.io, built from scratch using node.js. Voler.io was an online multiplayer game in which each player plays as a rocket orbiting a planet. Players must adjust their thrust so that they avoid other players and the planet. Currently a work in progress.
    • GitHub
  • BubbleLock
    • A widget for iPhone lock screens that would, when charging, show a battery (filled to the appropriate level) with bubbles floating up to the battery.
    • Example in use
  • Hedron
    • A widget for iPhone lock screens that would show an interactive shape on the lockscreen. The attached example video will demonstrate this. Multiple different shapes were supported.
    • Example in use
  • PHP Photo Album for Rowing Club
    • A photo album script written in PHP for my local rowing club. Dynamically generates pages for all folders (and recursively for subfolders) in the photo directory of the site's webserver. Currently in use at http://mcsbc.co.uk, however is password-protected for privacy reasons.
    • GitHub (Coming Soon)
  • Google Search Widget
    • For this project I collaborated with a graphical designer in order to make a Google search bar for the homescreens of rooted iOS devices. I worked on the Javascript behind the widget, including linking the widget with an existing package that handled Google voice searches (using Cycript).
    • Repository Link
  • Dweets
    • Dwitter is a site which allows users to generate canvas animations with the restriction of only being able to use 140 characters. This can be quite entertaining! My account is linked below.
    • Dwitter Account
